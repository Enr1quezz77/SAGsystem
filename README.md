# SAGsystem
 Sistema Gerencial de Control de Asistencia Biométrico desarrollado en PHP, JavaScript y SQL.
# 📊 S.A.G - Sistema Automatizado de Gestión 

Este es el proyecto de grado principal para optar al título de T.S.U. en Informática. Consiste en un sistema comercial e institucional diseñado desde cero para centralizar la gestión de personal, control de asistencia en tiempo real con hardware biométrico, administración de nóminas y auditorías operativas.

---

## 🎨 Diseño de Interfaz (UI/UX)
El producto cuenta con una experiencia de usuario optimizada y moderna, planificada inicialmente en **Figma** mediante wireframes y prototipos de alta fidelidad, garantizando un entorno limpio y equilibrado para el personal administrativo:
* **Módulo de Autenticación:** Interfaz dividida con diseño responsive e intuitivo.
* **Panel de Administrador:** Dashboard centralizado con contadores analíticos dinámicos y paneles informativos de asincronía.

---

## 🛠️ Tecnologías y Arquitectura Utilizadas
El software aplica una arquitectura cliente-servidor sólida y modular:
* **Backend:** PHP para la lógica de negocio, control de sesiones seguro y procesamiento de datos.
* **Frontend:** HTML5, JavaScript (ES6+) y **Tailwind CSS** para un entorno visual altamente optimizado y estilizado.
* **Asincronía:** Implementación de peticiones asíncronas mediante **AJAX** para flujos de datos rápidos en la lectura de biométricos y estados de conexión del sistema.
* **Base de Datos:** SQL relacional modelada exhaustivamente para manejar registros de usuarios, empleados, nóminas, asistencias, inasistencias y reportes.
* **Reportes:** Módulo dedicado para la exportación de reportes de asistencia directamente a formato PDF.

---

## 🚀 Funcionalidades Clave del Sistema
* 👥 **Gestión Completa de Empleados y Usuarios:** Clasificación de roles, privilegios y perfiles del personal.
* 🕒 **Asistencias e Inasistencias en Tiempo Real:** Monitorización activa de ingresos y egresos diarios.
* 📑 **Módulos Administrativos Avanzados:** Control integrado de amonestaciones, permisos justificados, cierres de caja detallados y auditorías operativas del sistema.
